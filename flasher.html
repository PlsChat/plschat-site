PLSChat Web Flasher

Flash the latest PLSChat firmware directly to your Heltec WiFi LoRa 32 (V4) from your browser.
No software installation required.

📋 How to Flash:

1. Plug in your Heltec V4 via USB
2. Hold the BOOT button (small button next to USB-C)
3. Click "Install" or "Connect"
4. Release BOOT button when it starts uploading
5. Wait for completion (green success message)

Works best on Chrome / Edge. Firmware includes licensed repeater mode.
