🏭 Company Firmware — v007

Tamper-enabled build • ESP32-S3

Flash the v007 firmware to a company device. Hold the BOOT button on the Heltec while clicking Connect in the installer. After flashing, provision it with COMPANY:ON over serial to enable the tamper module.

Note: this flashes whatever /manifest-v007.json points to. Once a device has secure boot + flash encryption enabled (Release mode), it can no longer be reflashed here — that's expected. Clicking Connect automatically releases the Licence Manager's serial connection first, so you don't need to refresh between sections.

🔒 Company Mode & Tamper

Serial • v007 • persisted to NVS

Fresh v007 devices ship with the tamper module off (companyDevice = false). Enabling Company Mode sends the exact command COMPANY:ON over serial, which arms the case-lid / USB tamper watch and saves the flag to NVS so it survives reboots. Use Disable (COMPANY:OFF) to turn it back off.

Only enable this on owner-deployed company units. With tamper armed, opening the case (or, if VBUS-trip is on, attaching USB) will lock or wipe that device's own keys per its policy. Set the tamper policy first if you need WIPE vs LOCK.

Uses the same USB serial port as the Licence Manager. If nothing is connected yet, your browser will prompt you to pick the device. Watch for COMPANY_OK in the status line above.
