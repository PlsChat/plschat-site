Your licence is a short signed string in the form v1|MAC|expiry|tier|signature. The signature is created with an Ed25519 private key — the same cryptography used to secure SSH and modern TLS — that lives only on our server and is never shipped in any file, web page, or device firmware.

It can't be forged. Change a single character and the signature no longer matches — your device and our server both reject it. Making a valid licence requires the private key, which no one else has.

It's bound to your device. Each licence is tied to one device's MAC address. Even if someone copies it, it won't activate on any other device — the firmware checks the MAC before accepting it.

Your payment is verified on-chain. A licence is only issued after your PLSC payment to the treasury is confirmed by two independent PulseChain nodes, and each transaction can be used exactly once.

Your privacy is yours to keep. We never ask for your name or email. For maximum privacy, shield your PLSC through LibertySwap or ZKX Wallet and pay from a fresh wallet.

PLSChat firmware is open source. Because you fully own your hardware, a technically capable owner can always modify the code running on their own device — this is true of any open device and is by design. It doesn't affect anyone else's licence or let anyone create a valid licence they didn't pay for.
